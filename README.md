## 👨‍💻 About

This is the personal portfolio website of **Abidul Hasan Amin**, a CSE Engineering Student from Bangladesh. The portfolio showcases his skills, projects, achievements, and community involvement.

---

## ✨ Features

- ⚡ Fully responsive design (mobile + desktop)
- 🌧️ Matrix rain background animation
- 🔮 Floating particle effects
- 🖼️ Lightbox image viewer (click on photo)
- 🎯 Smooth scroll navigation with hover effects
- 📜 Scroll reveal animations
- 🔗 All social & community links connected
